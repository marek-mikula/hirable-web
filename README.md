# Alpacca FE client

FE client for Alpacca learning management system.

## z-index list

| Depth | Objects                                        |
|-------|------------------------------------------------|
| 10    | selects, dropdowns, table head, badge indicators |
| 20    | table loader                                   |
| 30    | table head, table foot                         |
| 40    | sticky navbar                                  |
| 50    | sidebars (notification + menu)                 |
| 100   | modal windows                                  |
| 125   | tooltips                                       |
| 150   | toast messages                                 |
